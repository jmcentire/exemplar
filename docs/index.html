Exemplar

Governed code review — reference implementation for the FOSS governance stack

What

Exemplar is a CLI and MCP tool that accepts pull request diffs, routes them through multi-stage analysis (security, correctness, style, architecture), and produces verified review reports with full audit trails. Every stage is governed by the FOSS governance stack — not as a demo, but as the product itself. The governance IS the architecture.

The Stack

ConstrainSpecification before code — interviews, boundaries, artifacts
PactContract-first decomposition, typed interfaces, hidden tests

ChroniclerEvent collection and story assembly from stage boundaries

TransmogrifierRegister-aware prompt translation — normalize input for optimal LLM output
